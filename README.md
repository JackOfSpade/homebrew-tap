# JackOfSpade's Homebrew Tap

Personal Homebrew tap for custom formulae.

## Installation

Tap this repository:

```bash
brew tap jackofspade/tap
```

## Available Formulae

| Name | Description | Installation |
| --- | --- | --- |
| [`ccblocks`](https://github.com/JackOfSpade/ccblocks) | time-shift Claude sessions to match your working hours | `brew install jackofspade/tap/ccblocks` |
